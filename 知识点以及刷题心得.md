## 一、 数据结构 

### 1. 树状数组 (BIT)
树状数组是一种基于二进制思想的数据结构，其基本用途是**维护序列的前缀和**。
对于给定的序列 $a$，我们建立一个数组 $c$，其中 $c[x]$ **保存序列 $a$ 的区间 $[x-\text{lowbit}(x)+1, x]$ 中所有数的和**。

> **前置知识：lowbit 定义**
> `lowbit(n)` 定义为非负整数 $n$ 在二进制表示下“最低位的 1 及其后边所有的 0”。

```cpp
// lowbit 求法
int lowbit(int x) { return x & (-x); }
```

**树状数组核心模板：**
```cpp
struct BIT {
    int n;
    vector<int> tree;
    BIT(int n) : n(n), tree(n + 1, 0) {}
    
    int lowbit(int x) { return x & -x; }
    
    // 单点增加
    void add(int x, int d) {
        for (int i = x; i <= n; i += lowbit(i)) tree[i] += d;
    }
    
    // 查询前缀和 [1, x]
    int ask(int x) {
        int sum = 0;
        for (int i = x; i > 0; i -= lowbit(i)) sum += tree[i];
        return sum;
    }
};
```

### 2. 线段树
支持区间修改以及区间查询的高阶数据结构

听完线段树1的讲解后本人认为线段树的Node构造需要遵循

>首先l与r是必须的，其次就是答案让查询的东西，然后我们思考这个东西是否能被递归pushup更新上去，如果不能我们就继续把这个元素拆分成可以递归更新的属性，这便是框架

大致模板如下：

```cpp
//由于本人喜好结构体类型格式
struct SegTree {

	struct Node {
		int l, r;//基础的是l,r，去存储基本的区间
	} tr[N * 4];

	vector<int> w;
//原数组的读入
	void Sread(int n) {
		w.push_back(0);
		fors(i, 1, n) {
			int num;
			cin >> num;
			w.pb(num);
		}
	}

	void pushup(Node &u, Node& l , Node&r) {
		//负责细节上的修改
	}

	void pushup(int u) {
		pushup(tr[u], tr[u << 1], tr[u << 1 | 1]);
		//广义对根节点的向上修改
	}

	void build(int u, int l, int r) {
		//建造线段树
		if(l==r) tr[u]={};//括号初始化，叶子节点
		else{
             tr[u].l=l,tr[u].r=r;
			int mid = (l+r)>>1;
			build(u << 1, l , mid),build(u << 1 | 1 , mid+1,r);
			pushup(u);
		}
	}

	void modify(int u, int x, int v) {
		if (tr[u].l == x && tr[u].r == x) {}//叶子节点的修改
		else {
			int mid = (tr[u].l + tr[u].r) >> 1;
			if (mid >= x) modify(u << 1, x, v);
			else modify(u << 1 | 1, x, v);
			pushup(u);
		}
	}

	Node query(int u, int l, int r) {
		if (l > r) return{0};
		if (tr[u].l >= l && tr[u].r <= r) return tr[u];//如果该线段被包含在了那个区间，我们就纳入
		else {
			int mid = (tr[u].l + tr[u].r) >> 1;
			if (r <= mid) return query(u << 1, l, r);//如果都在左边
			else if (l > mid) return query(u << 1 | 1, l, r);//都在右边
			else {
				auto left = query(u << 1, l, r);//左边的情况
				auto right = query(u << 1 | 1, l, r);//右边的情况
				Node res;
				pushup(res, left, right);//pushup直接修改结果
				return res;
			}
		}
	}

};
```

```cpp
struct List {

		init();
	}

	void init() {
		idx = 2;
		r[0] = 1;

		e[idx] = x;
		l[r[k]] = idx;
		r[idx] = r[k];
		l[idx] = k;
		r[k] = idx;
		idx++;
	}

		insert_right(0, x);
	}

		insert_right(l[1],x);
	}

		r[l[k]] = r[k];
		l[r[k]] = l[k];
	}

		insert_right(l[k], x);
	}


};
```

## 二、 算法思想与策略 

### 1. 二维偏序（扫描线降维打击）
**核心本质**：利用“排序”消灭一个维度，再用数据结构（BIT）动态维护另一个维度。

**通俗理解（HR 招聘模型）**：
假设求职者有两个属性：**笔试成绩 ($X$)** 和 **面试成绩 ($Y$)**。要求找出 $X \le 80$ 且 $Y \le 90$ 的人。
直接嵌套循环会导致 $O(N^2)$。我们的降维魔法如下：
1. **消灭第一维 $X$**：将所有人按笔试成绩 $X$ 从低到高排队。现在我们从左到右挨个审核，审核到第 $k$ 个人时，前面的 $k-1$ 个人 $X$ 必然满足条件！这样 $X$ 维就被**排序动作**消灭了。
2. **维护第二维 $Y$**：准备一个一维的树状数组（黑板）专门记录面试成绩 $Y$。每审核过一个人就把他的 $Y$ 登记上去。查询时，直接在这个一维黑板上求 $Y$ 的前缀和即可。

### 2. 离散化 
当坐标值域极大（如 $10^9$）但元素个数有限（如 $10^5$）时，将数值映射为 $1 \sim N$ 的紧凑排名，以便存入树状数组。

**离散化黄金三连：复制 -> 排序+去重 -> 二分找排名**
```cpp
vector<int> b = a; 
sort(b.begin(), b.end()); 
b.erase(unique(b.begin(), b.end()), b.end()); 

for (int i = 0; i < n; i++) {
    // lower_bound 返回迭代器，减去 begin() 得到 0-based 下标。加 1 变成 BIT 1-based 排名
    a[i] = lower_bound(b.begin(), b.end(), a[i]) - b.begin() + 1; 
}
```

### 3. 归并排序求逆序对
注意右半段较小时才会产生逆序对。

```cpp
int a[N], tmp[N]; 

int merge_sort(int l, int r) {
    if (l >= r) return 0;

    int mid = l + (r - l) / 2; // 【防爆排雷】：不要写 l + r >> 1，注意优先级和防溢出
    int res = merge_sort(l, mid) + merge_sort(mid + 1, r);

    int i = l, j = mid + 1, k = 0;
    while (i <= mid && j <= r) {
        if (a[i] <= a[j]) {
            tmp[k++] = a[i++]; 
        } else {
            // 右边较小，构成逆序对！
            // 因为左半段 [i, mid] 是有序的，a[i] 到 a[mid] 都大于 a[j]
            res += mid - i + 1; 
            tmp[k++] = a[j++];
        }
    }

    while (i <= mid) tmp[k++] = a[i++];
    while (j <= r) tmp[k++] = a[j++];
    for (i = l, k = 0; i <= r; i++, k++) a[i] = tmp[k];

    return res;
}
```

### 4. 懒惰更新与时间戳
面对高频的全局覆盖（`a = d`）操作，绝对不写 $O(n)$ 的硬拷贝。
* **策略**：维护一个全局 `last_reset` 和单点的 `last_change[i]`。
* **致命陷阱**：在对任何数据进行判断之前，必须先校验它的时间戳！如果过期了，必须先“唤醒”恢复真实值，用假数据做判断是 Lazy Update 的大忌。

### 5.数学公式的合理拆分
当你推理出来本题目的数学公式的时候，应当往下一直化简。
* **思想** ： 分离包含有多个变量的项，使公式中不同变量之间互相独立

例子： 
> $\sum_{i=1}^{x}\sum_{j=1}^{i} b_j = \sum_{i=1}^{x}(x-i+1)*b_i$
>
> $\sum_{i=1}^{x}(x-i+1)*b_i = (x+1)\sum_{i=1}^{x}b_i-\sum_{i=1}^{x}i*b_i$
> 
这样一来，该数学式子就可以拆分成两个前缀和，都可以实现

### 1. 拆环成链
**核心思想**：如何简洁地表示环中两点距离？开一个二倍数组，将环拆成链子。
`pre[n+i]` 的含义是从起点出发走两圈到该点。例如：顺时针 6 点到 4 点的距离 = **走两圈到 4 点的距离 - 走一圈到 6 点的距离**。

```cpp
// 核心逻辑：s 为第一圈的总周长
for (int i = 1; i <= n; i++) {
    pre[i + n] = pre[i] + s; 
}
```

### 2. DFS / 回溯法
DFS 虽然千变万化，但万变不离其宗。核心在于**保存现场与恢复现场**。

```cpp
void dfs(int step) {
    if (/* 到达终点 或 满足边界条件 */) {
        // 记录答案或输出
        return;
    }
    
    for (/* 遍历当前状态的所有可能选择 */) {
        if (/* 该选择合法且未被访问 */) {
            // 1. 标记该选择已被访问
            // 2. 保存现场 (应用该选择)
            
            dfs(step + 1); // 递归进入下一层
            
            // 3. 恢复现场 (撤销该选择) —— 回溯的核心！
            // 4. 解除访问标记
        }
    }
}
```

## 四、 数学与数论 

### 1. 取模极限规律 
**核心定理**：当 $B < A$ 时，对于给定的整数 $A$ 对任意正整数 $B$ 取模，其最大可能取值为：
**$A \bmod B \le \lfloor \frac{A-1}{2} \rfloor$**
*(注：当且仅当 $B = \lfloor \frac{A}{2} \rfloor + 1$ 时取到最大值。如果 $B > A$，则 $A \bmod B = A$)*。

### 2. MEX 
指在一个非负整数集合中**不包含的最小非负整数**。
* 例：集合 $S = \{0, 1, 2, 3\}$，那么 $\operatorname{mex}(S) = 4$。

### 3. 连续整数异或和的周期性质
定义前缀异或和 $P(i) = 1 \oplus 2 \oplus 3 \oplus \dots \oplus i$，规定 $P(0) = 0$。
$P(i)$ 的取值具有以 $4$ 为周期的强规律：
* $P(i) = i$，若 $i \equiv 0 \pmod 4$
* $P(i) = 1$，若 $i \equiv 1 \pmod 4$
* $P(i) = i+1$，若 $i \equiv 2 \pmod 4$
* $P(i) = 0$，若 $i \equiv 3 \pmod 4$

### 4. 哥德巴赫猜想

哥德巴赫猜想的现代陈述为：**任一大于5的整数都可写成三个质数之和。**

由此可以得到两种情况：

1.强哥德巴赫猜想（强哥猜）：**即任一大于2的偶数都可写成两个质数之和**；（未被证明，但是同时也没有被推翻，即在本体的范围内强哥猜成立）

2.弱哥德巴赫猜想（弱哥猜）**:任何一个大于5的奇数都能被表示成三个奇质数的和**。（一个质数可以被多次使用）;（已经被证明）

## 五、 考场通关与刷题心法 

- [ ] **翻译题目**：强迫自己用一句话、严谨的数学语言重新描述题意，剥离冗余背景。
- [ ] **极限施压**：要想出能卡掉程序的**极限数据与边界**，穷举所有极端情况，防止漏判。
- [ ] **拒绝干瞪眼**：遇到抽象限制，必须**纸笔手工模拟**，自己去造数据理解流程。
- [ ] **复杂度反推**：从数据范围反推时间复杂度（例如看到 $10^5$ 盲猜 $O(N \log N)$），再去对应层级的算法和数据结构里找武器。
- [ ] **避坑指南**：字符串哈希容易冲突且极难调试，能用 `std::map` 转化为数字 ID 的，就优先转成数字 ID！
- [ ] **查询越界思维**：用 `upper_bound - begin()` 求 $\le target$ 的个数时，算出来的本质是“越界者的排名”，直接用 `ask(rank - 1)` 查询，逻辑闭环最严密。

## 六、 矩阵与坐标技巧

**主副对角线恒等式：**

- 主对角线（左上-右下）：$i - j = \text{const}$。
- 副对角线（右上-左下）：$i + j = \text{const}$。

**坐标转换应用：** 在 `X-Sum` 题中，利用此规律可将 $O(N^2)$ 的动态搜索优化为 $O(1)$ 的预处理读取。

### 【实战例题：配对统计】完整扫描线代码
```cpp
#include <bits/stdc++.h>
#define int long long
using namespace std;
const int N = 3e5+10;

int test, n, m;
int f[N];

void add(int x, int y) {
	for (; x <= n; x += x & (-x)) f[x] += y;
}

int ask(int x) {
	int res = 0;
	for (; x; x -= x & (-x)) res += f[x];
	return res;
}

void solve() {
	cin >> n >> m;
	if (n == 1) {
		cout << "0\n";
		return;
	}
	vector<pair<int, int>> a(n + 1);
	for (int i = 1; i <= n; i++) {
		cin >> a[i].first;
		a[i].second = i;
	}

	sort(a.begin() + 1, a.end(), [](const auto & aa, const auto & bb) {
		return aa.first < bb.first;
	});

	vector<pair<int, int>> pairs;

	// 处理首尾情况
	int l = min(a[1].second, a[2].second), r = max(a[1].second, a[2].second);
	pairs.push_back({l, r});
	l = min(a[n].second, a[n - 1].second), r = max(a[n].second, a[n - 1].second);
	pairs.push_back({l, r});

	for (int i = 2; i < n; i++) {
		int ldif = a[i].first - a[i - 1].first;
		int rdif = a[i + 1].first - a[i].first;
		if (ldif < rdif) {
			int x = a[i].second, y = a[i - 1].second;
			if (x > y) swap(x, y);
			pairs.push_back({x, y});
		} else if (ldif == rdif) {
			int x1 = a[i].second, y1 = a[i - 1].second;
			int x2 = a[i].second;
			int y2 = a[i + 1].second;
			if (x1 > y1) swap(x1, y1);
			if (x2 > y2) swap(x2, y2);
			pairs.push_back({x1, y1});
			pairs.push_back({x2, y2});
		} else {
			int x = a[i].second, y = a[i + 1].second;
			if (x > y) swap(x, y);
			pairs.push_back({x, y});
		}
	}

	sort(pairs.begin(), pairs.end(), [](const auto & aa, const auto & bb) {
		if (aa.second != bb.second) return aa.second < bb.second;
		return aa.first < bb.first;
	});

	int ans = 0, pair_idx = 0;
	int total_pairs = pairs.size();

	vector<pair<int, pair<int, int> > > dicar;
	for (int i = 1; i <= m; i++) {
		int l, r;
		cin >> l >> r;
		dicar.push_back({i, {l, r}});
	}

	sort(dicar.begin(), dicar.end(), [](const auto & aa, const auto & bb) {
		return aa.second.second < bb.second.second;
	});

    // 扫描线核心逻辑
	for (auto &it : dicar) {
		while (pair_idx < total_pairs && pairs[pair_idx].second <= it.second.second) {
			add(pairs[pair_idx].first, 1);
			pair_idx++;
		}
		int lef = ask(it.second.first - 1);
		int cnt = pair_idx - lef;
		ans += cnt * it.first;
	}

	cout << ans << '\n';
}

signed main() {
	ios::sync_with_stdio(0); cin.tie(0);
	solve();
	return 0;
}
```
