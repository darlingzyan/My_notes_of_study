# `第一题数列分段`

#### 大概题意

给定一个长度为n的数组，要求将数组分为连续的m段，使得每段的最大值最小。

### 做法

因为是最大值最小问题，很容易想到二分，我们可以二分每段的最大值是多少，从而将n个数进行分段处理，如果当前段的和大于我们二分的值则不符合题意，应该在这个位置进行分段，否则将当前位置加到前一段当中去。时间复杂度为O(N*log(∑i=1Nai   ∑*i*=1*N**a**i*))详细请看代码。

#### 注意点：

- 数据范围很大要开long long。
- 输入很多如果使用cin，cout的话需要关闭同步流。
- 注意上下界问题。

```cpp

#define int long long
typedef tuple<int,int,int> tp;

constexpr int N=500010,mod=1e9+7,inf=1e9;
constexpr double pi=3.1415926535897932384626;

#define get_count(x) __builtin_popcount(x)

    int __=1;
//    cin >> __;

        int n,m;
        cin >> n >> m;
        vector<int> a(n+1);
        int r=0,l=0;
        for(int i=1;i<=n;i++) cin >> a[i],r+=a[i],l=max(l,a[i]);
        auto check=[&](int k)->bool{
            int cnt=1,s=0;
            for(int i=1;i<=n;i++){
                if(s+a[i]<=k){
                    s+=a[i];
                }
                else cnt++,s=a[i];
            }
            return cnt<=m;
        };
        while(l<r){
            int mid=l+r>>1;
            if(check(mid)) r=mid;
            else l=mid+1;
        }
        cout << r << '\n';
    }
     // system("color 04");

}

# `第二题 约瑟夫问题`

### 大概题意

n个人围成一个圈，然后依次数数，当数到m的时候，该人出圈并且输出。

### 做法

因为数据范围很小，我们可以直接模拟，开一个标记数组记录每个人是否出圈，依次模拟直至全部出圈为止,最坏时间复杂度为O(N2*N*2)。 第二种做法是公式法 详细看洛谷题解区。

# `第三题 求解迷宫从入口到出口的一条最短路径`

### 大概题意

从(1,1)位置按照向右、向下、向左、向上的顺序向前搜索直到(n-2,n-2)位置，找一条最短路径

### 做法

bfs适用于求解最短，所以我们可以直接bfs，然后开一个map记录一下，走到每个点时候的前驱结点，最后用栈的方式找到路径。因为每个点最多只会遍历一次，所以时间复杂度为O(N2*N*2)。

```
#include<bits/stdc++.h>
using namespace std;
typedef long long ll;
mt19937 rnd(time(0));
#define int long long
typedef tuple<int,int,int> tp;
#define x first
#define y second
typedef pair<int,int> pii;
typedef pair<double,double> pdd;
constexpr int N=500010,mod=1e9+7,inf=1e9;
constexpr double pi=3.1415926535897932384626;
const ll  P=rnd()%mod;
#define all(a) a.begin(),a.end()
#define get_count(x) __builtin_popcount(x)
int dx[]={0,1,0,-1,1,1,-1,-1,0};
int dy[]={1,0,-1,1,1,-1,-1,1,0};

signed main(){
    cin.tie(nullptr)->sync_with_stdio(false);
    int __=1;
//     cin >> __;
    while(__--){
        int n;
        cin >> n;
        vector<vector<int>> g(n+1,vector<int>(n+1)),vis(n+1,vector<int>(n+1));
        map<pii,pii> path;
        for(int i=1;i<=n;i++){
            for(int j=1;j<=n;j++){
                cin >> g[i][j];
            }
        }
        queue<pii> q;
        q.push({2,2});
        vis[2][2]=1;
        int ok=0;
        while(q.size()){
            auto [x,y]=q.front();
            q.pop();
            if(x==n-1&&y==n-1){
                ok=1;
                break;
            }
            for(int i=0;i<4;i++){
                int a=dx[i]+x,b=dy[i]+y;
                if(a<1||a>n||b<1||b>n) continue;
                if(g[a][b]||vis[a][b]) continue;
                path[{a,b}]={x,y};
                vis[a][b]=1;
                q.push({a,b});
            }
        }
        if(ok){
            stack<pii> st;
            int x=n-1,y=n-1;
            while(x!=2||y!=2){
                st.push({x,y});
                auto [a,b]=path[{x,y}];
                x=a,y=b;
            }
            st.push({x,y});
            while(st.size()){
                auto [x,y]=st.top();
                cout << "(" << x-1 << "," << y-1 << ")";
                st.pop();
            }
            cout << '\n';
        }
        else cout << "NO\n";
    }
    system("color 04");
    return 0;
}

# `第四题 任务拓扑排序`

### 大概题意

如果想要完成任务a，则先要完成任务b。问是否找到一个任务解决次序，使得可以完成所有任务.那么我们就要连一条从b到a的边。所以说在图中找到一个拓扑序即可。

### 做法

拓扑排序,如果想要完成任务a，则先要完成任务b,那么我们就要连一条从b到a的边。所以说在图中找到一个拓扑序即可。时间复杂度为O(（N+M*N*+*M*）* log) 后面的log来源于优先队列，常数较大。

#### 注意点：

要求找字典序最小的一个次序，普通的拓扑排序是用队列实现的，现要求最小的一个字典序，我们可以用一个小根堆实现，STL的优先队列，priority_queue<T,vector<T>,greater<T>> q

```
#include<bits/stdc++.h>
using namespace std;
typedef long long ll;
mt19937 rnd(time(0));
#define int long long
typedef tuple<int,int,int> tp;
#define x first
#define y second
typedef pair<int,int> pii;
constexpr int N=500010,mod=1e9+7,inf=1e9;
constexpr double pi=3.1415926535897932384626;
const ll  P=rnd()%mod;
#define all(a) a.begin(),a.end()
#define get_count(x) __builtin_popcount(x)
int dx[]={1,-1,0,0,1,1,-1,-1,0};
int dy[]={0,0,1,-1,1,-1,-1,1,0};
vector<int> e[N];
int d[N];
signed main(){
    cin.tie(nullptr)->sync_with_stdio(false);
    int __=1;
//    cin >> __;
    while(__--) {
        int n,m;
        cin >> n >> m;
        for(int i=1;i<=m;i++){
            int a,b;
            cin >> a >> b;
            e[a].push_back(b);
            d[b]++;
        }
        priority_queue<int,vector<int>,greater<int>> q;
        vector<int> ans;
        for(int i=0;i<n;i++){
            if(!d[i]){
                q.push(i);
            }
        }
        while(q.size()){
            auto u=q.top();
            q.pop();
            ans.push_back(u);
            for(auto j:e[u]){
                if(--d[j]==0){
                    q.push(j);
                }
            }
        }
        if(ans.size()==n){
            for(auto x:ans) cout << x << ' ';
            cout << '\n';
        }
        else{
            cout << "unworkable project" << '\n';
        }
    }
     system("color 04");
    return 0;
}

# `第五题 单调栈`

### 大概题意

n个人横向站成一队每个人的身高是ai*a**i*，如果俩个人之间的人的身高全部小于等于俩人则这俩人可以看到彼此，求有多少对可以互相看到的人。

### 做法

考虑每个人怎么看到他左边的人。

1. 左边第一个比他高的人肯定可以看见。
2. 他可以看见左边单调递减的所有人
   直接维护一个单调递减的单调栈即可。时间复杂度为O（N）。

#### 注意点：

对于同样身高的人，我们可以把他们合并成一个并累计贡献，所以采用stack<pair<int,int>> st，第一个位置存身高，第二个位置存这一块的身高有几个人，具体看代码实现。

```
#include<bits/stdc++.h>
using namespace std;
typedef long long ll;
mt19937 rnd(time(0));
#define int long long
typedef tuple<int,int,int> tp;
#define x first
#define y second
typedef pair<int,int> pii;
constexpr int N=500010,mod=1e9+7,inf=1e9;
constexpr double pi=3.1415926535897932384626;
const ll  P=rnd()%mod;
#define all(a) a.begin(),a.end()
#define get_count(x) __builtin_popcount(x)
int dx[]={1,-1,0,0,1,1,-1,-1,0};
int dy[]={0,0,1,-1,1,-1,-1,1,0};
signed main(){
    cin.tie(nullptr)->sync_with_stdio(false);
    int __=1;
//    cin >> __;
    while(__--){
        int n;
        cin >> n;
        stack<pii> st;
        int ans=0;
        for(int i=1,x;i<=n;i++){
            cin >> x;
            pii res={x,1};
            while(st.size()&&st.top().x<=x){
                ans+=st.top().y;
                if(st.top().x==x) res.y+=st.top().y;
                st.pop();
            }
            if(st.size()) ans++;
            st.push(res);
            cout << ans << " \n"[i==n];
        }
        cout << ans << '\n';
    }
     system("color 04");
    return 0;
}

# `第六题 单调栈`

### 大概题意

给定n个数，然后在所有连续子序列里面找到最大值和最小值的差，然后进行一个累加求和。

### 做法

我们可以考虑对于每个数ai*a**i*作为最大值他所能影响的区间有哪些，很明显，对于i左边找到第一个l位置，使得 al>ai  *a**l*>*a**i*, 同样在右边找到第一个r位置，使得 ar>ai  *a**r*>*a**i*。 此时ai作为最大值的贡献为 ai∗(i−l)∗(r−i)*a**i*∗(*i*−*l*)∗(*r*−*i*) 很明显我们上面的操作可以用单调栈来实现。 同理最小值的贡献也是如此，最终答案就是最大值的贡献减去最小值的贡献。时间复杂度为O（N）

#### 注意点

为了避免区间重复计算 我们只找左边第一个大于或者小于ai*a**i*的位置，右边只找第一个大于等于或者小于等于ai*a**i*的位置，详细部分请看代码。

       int n;
       cin >> n;
       vector<int> a(n+1);
       for(int i=1;i<=n;i++) cin >> a[i];
       vector<int> l(n+1),r(n+1);
       // 先求最大值 找左边第一个大于a[i]的数
       stack<int> st;
       for(int i=1;i<=n;i++){
           while(st.size()&&a[st.top()]<=a[i]) st.pop();
           if(!st.size()) l[i]=1;

       int ans=0;
       for(int i=1;i<=n;i++)

//        for(int i=1;i<=n;i++) cout << l[i] << " \n"[i==n];
       // 找右边第一个小于a[i]的数

//        for(int i=1;i<=n;i++) cout << r[i] << " \n"[i==n];
       for(int i=1;i<=n;i++)
           ans-=(i-l[i]+1)*(r[i]-i+1)*a[i];
       cout << ans << '\n';

# `第七题 最长对称子串`

### 大概题意

找最长回文子串

### 做法一

二分+哈希 对于每一个位置i以他为中心，二分回文子串长度，因为回文子串有俩种，一种是奇数串，一种是偶数串，所以要二分俩次，详细看代码。时间复杂度O(N∗log(N)*N*∗*l**o**g*(*N*))

```
#include<bits/stdc++.h>
using namespace std;
typedef long long ll;
mt19937 rnd(time(0));
#define int long long
typedef tuple<int,int,int> tp;
#define x first
#define y second
typedef pair<int,int> pii;
constexpr int N=200010,mod=1e9+7,inf=1e18;
constexpr double pi=3.1415926535897932384626;
const ll  P=rnd()%mod;
#define all(a) a.begin(),a.end()
#define get_count(x) __builtin_popcount(x)
int dx[]={1,-1,0,0,1,1,-1,-1,0};
int dy[]={0,0,1,-1,1,-1,-1,1,0};
int hl[N],hr[N],p[N];
signed main(){
    cin.tie(nullptr)->sync_with_stdio(false);
    int __=1;
//    cin >> __;
    while(__--){
        string s;
        getline(cin,s);
        int n=s.size();
        s=" "+s;
        p[0]=1;
        for(int i=1,j=n;i<=n;i++,j--){
            hl[i]=(hl[i-1]*P+s[i]+mod)%mod;
            hr[j]=(hr[j+1]*P+s[j]+mod)%mod;
            p[i]=p[i-1]*P%mod;
        }
        auto check=[&](int l1,int r1,int l2,int r2)->bool{
            int len=r1-l1+1;
          if(l1<1||l2<1||r1>n||r2>n||len<1) return false;
          return (hl[r1]-hl[l1-1]*p[r1-l1+1]%mod+mod)%mod==(hr[l2]-hr[r2+1]*p[r2-l2+1]%mod+mod)%mod;
        };
        int ans=1;
        for(int i=1;i<=n;i++){
            int l=0,r=n;
            while(l<r){
                int mid=l+r+1>>1;
                if(check(i-mid,i-1,i+1,i+mid)) l=mid;
                else r=mid-1;
            }
            ans=max(ans,r*2+1);
            l=0,r=n;
            while(l<r){
                int mid=l+r+1>>1;
                if(check(i-mid+1,i,i+1,i+mid)) l=mid;
                else r=mid-1;
            }
            ans=max(ans,r*2);
        }
        cout << ans << '\n';
    }
    return 0;
}
```

### 做法二

Manache算法，俗称马拉车算法。可以O(N)的求最大回文子串，代码如下。

```
#include<bits/stdc++.h>
using namespace std;
int main()
{
	string str;
	getline(cin,str);
	string s="";
	// 将子串s转换成Manacher串 
	for(int i=0;i<str.size();i++)
	{
		s+='#';
		s+=str[i];
	}
	s+='#';
//	cout<<s<<endl;
	int arr[s.size()];
	int R=-1;
	int C=-1;
	int maxx=INT_MIN;
	for(int i=0;i<s.size();i++)
	{
		// 以i为中心至少扩的位置 2*c-1是i关于中心点C对称的点 
		arr[i]=R>i?min(arr[2*C-i],R-i):1;
		//右边和左边都在串内 
		while(i+arr[i]<s.size()&&i-arr[i]>-1)
		{
			//向外扩一个 回文半径+1
			if(s[i+arr[i]]==s[i-arr[i]]) arr[i]++;
			else break;
		} 
		// 如果 i+arr[i] > R 即 最右边界需要更新维护 
		if(i+arr[i]>R){
			R=i+arr[i];
			C=i; 
		}
		//记录最大长度 
		maxx=max(maxx,arr[i]);
	} 
	cout<<maxx-1<<endl;
    return 0;
}
```

### 做法四

动态规划 定义dp i到j 初始化i==j的时候 如果si=sj 则推断出dpij 需要i+1和j-1 即dpi+1 j-1 的最长长度 加2 不同 考虑i或者 j位置 则由 i位置到 j-1 或者由i+1到j位置 取最大。时间复杂度O(N2*N*2)

```
int main()
{
	string s;
	getline(cin,s);
	int n=s.size();
	int maxx=INT_MIN; 
	if(n==1) maxx=1;
	vector<vector<int>> dp(n,vector<int>(n));
	for(int i=0;i<n;i++) dp[i][i]=1;
	for(int i=n-1;i>=0;i--)
	{
		for(int j=i+1;j<n;j++)
		{
			if(s[i]==s[j]) dp[i][j]=dp[i+1][j-1]+2;
			else dp[i][j]=max(dp[i+1][j],dp[i][j-1]);
			maxx=max(maxx,dp[i][j]);
		}
	}
	cout<<maxx;
	return 0;
}
```

# `第八题 进击的奶牛`

### 大概题意

### 做法

kruskal算法，首先将图中的所有边按照权重从小到大进行排序，然后从最小权重的边开始，依次选择边并加入到生成树中，同时确保不形成回路，直至生成包含所有顶点的树为止。kruskal算法适用于加权连通图，并且可以处理权值相同的情况。它通过并查集来判断是否形成回路，从而避免在生成树中出现环。该算法的时间复杂度为O(ElogE)，其中E是边的数量。模板题不做过多赘述，详细见代码。

# `第十四题 最长公共子序列`

### 大概题意

给定T组查询,每次查询给定俩个字符串问他们的最长公共子序列的长度是多少。

### 做法

动态规划，f(i,j)，设第一个字符串为s，第二个字符串为t，f(i,j）表示s中前i个元素和t中前j个元素的最长公共子序列长度是f(i,j)。 如果s[i]==t[j]的话，则f[i][j]=f[i-1][j-1]+1。否则f[i][j]=max(f[i-1][j],f[i][j-1]),最终输出f[n][m]即为答案。时间复杂度为O(T∗n∗m*T*∗*n*∗*m*)，其中n和m为s和t的字符串长度。

            for(int i=1;i<=n;i++){
                for(int j=1;j<=m;j++){
                    if(s[i]==t[j]) f[i][j]=f[i-1][j-1]+1;
                    else f[i][j]=max(f[i-1][j],f[i][j-1]);
                }

    }

# `第十五题 合并石子`

### 大概题意

有n堆石子围成一个圈，每次可以选择相邻的俩堆石子进行合并，问怎么样合并可以得到最大值和最小值。

### 做法

动态规划之区间DP，首先我们可以把环形问题分解成链式结构，简而言之就是破环成链，最简单的就是开双倍的空间，在[n+1,2*n]复制[1,n]区间就可以完成破环成链的操作。定义f(i,j)为合并第i堆到第j堆石子的最大值为f(i,j)，我们可以第一维枚举合并的长度是多少，假如是len，然后我们枚举合并石子的左端点l和右端点r，再找一个分段点k，代表从[l,k]和[k+1,r]进行合并，合并的代价为pre[r]-pre[l-1],pre为a数组的前缀和，所以我们可以得到状态转移方程f[l][r]=max(f[l][r],f[l][k]+f[k+1][r]+pre[r]-pre[l-1),最小值同理。时间复杂度为O(n3*n*3)

```

using namespace std;
typedef long long ll;
mt19937 rnd(time(0));
#define int long long
typedef tuple<int,int,int> tp;
#define x first
#define y second
typedef pair<int,int> pii;
constexpr int N=500010,mod=1e9+7,inf=1e18;
constexpr double pi=3.1415926535897932384626;
const ll  P=rnd()%mod;
#define all(a) a.begin(),a.end()
#define get_count(x) __builtin_popcount(x)
int dx[]={1,-1,0,0,1,1,-1,-1,0};
int dy[]={0,0,1,-1,1,-1,-1,1,0};
signed main(){
    cin.tie(nullptr)->sync_with_stdio(false);
    int __=1;
//    cin >> __;
    int n;
    while(cin >> n) {
        vector<int> a(n*2+1),pre(n*2+1);
        vector<vector<int>> f(2*n+1,vector<int>(2*n+1,-inf)),g(2*n+1,vector<int>(2*n+1,inf));
        for(int i=1;i<=2*n;i++) f[i][i]=g[i][i]=0;
        for(int i=1;i<=n;i++){
            cin >> a[i];
            a[i+n]=a[i];
        }
        for(int i=1;i<=2*n;i++) pre[i]=pre[i-1]+a[i];
        for(int len=2;len<=n;len++){
            for(int l=1;l+len-1<=2*n;l++){
                int r=l+len-1;

                }
            }
        }

        for(int i=1;i<=n;i++){
            ans=max(ans,f[i][i+n-1]);
            res=min(res,g[i][i+n-1]);
        }
        cout << res << ' ' << ans << '\n';
    }

# `第十六题 最小环`

### 大概题意

找到一个环满足大于等于3个结点，并且权值和最小。

### 做法

floyd算法，枚举三个点i,j,k，其中k为i和j中间的结点。所以由i,j,k三点组成的环的值为w[i][k]+w[k][j]+d[i][j],其中w(i,j)为i到j的距离，d(i,j)为i到j的最短距离。时间复杂度O(N3*N*3).

```

#define y second

const ll  P=rnd()%mod;

#define get_count(x) __builtin_popcount(x)
int dx[]={1,-1,0,0,1,1,-1,-1,0};

constexpr int M=110;
int d[M][M],g[M][M];
signed main(){
    cin.tie(nullptr)->sync_with_stdio(false);
    int __=1;
    // cin >> __;

        int n, m;
        cin >> n >> m;
        for(int i=1;i<=n;i++){
            for(int j=1;j<=n;j++){
                d[i][j]=inf;
            }
        }
        for (int i = 1; i <= n; i++) d[i][i] = 0;
        for (int i = 1; i <= m; i++) {
            int a, b, c;
            cin >> a >> b >> c;
            d[a][b] = min(d[a][b], c);
            d[b][a] = min(d[b][a], c);
        }
        memcpy(g, d, sizeof g);
        int ans = inf;
        for (int k = 1; k <= n; k++) {
            for (int i = 1; i < k; i++) {
                for (int j = i + 1; j < k; j++) {
                    ans = min(ans, d[i][j] + g[i][k] + g[k][j]);
                }
            }
            for (int i = 1; i <= n; i++) {
                for (int j = 1; j <= n; j++) {
                    d[i][j]=min(d[i][j],d[i][k]+d[k][j]);
                    d[j][i]=d[i][j];
                }
            }
        }

# 第十七题 Dijkstra算法求最短路

### 大概题意

把题目给的边连起来，然后问能否从1走到n，求一个最短距离，如果不能输出-1

### 做法

Dijkstra算法，有俩种，一种是朴素版，时间复杂度O(N2*N*2),另一种是堆优化版，时间复杂度O(M∗log(N))*M*∗*l**o**g*(*N*))。算法思路采用贪心了贪心的思路，基本思路是从一个节点开始，‌逐步构建出到达图中所有其他节点的最短路径。‌

Dijkstra算法是一种用于解决带权图中单源最短路径问题的算法，‌由荷兰计算机科学家Edsger W. Dijkstra于1959年提出。‌该算法通过逐步构建从起点到图中所有其他节点的最短路径，‌采用贪心策略，‌每次选择当前未处理节点中距离起点最近的节点进行处理，‌并更新其相邻节点的距离。‌算法的主要步骤包括：‌

初始化：‌设定起点，‌初始化所有节点到起点的距离，‌通常将起点到自身的距离设为0，‌其他节点的距离设为无穷大。‌ 选择未处理节点中距离起点最近的节点：‌在未处理的节点中选择距离起点最近的节点，‌并将其标记为已处理。‌ 更新相邻节点的距离：‌对于已选择的节点，‌更新其相邻节点的距离。‌如果通过已选择的节点到达某个节点的距离比原来记录的距离短，‌则更新该节点的距离。‌ 重复上述步骤：‌直到所有节点都被处理完毕，‌此时得到的每个节点到起点的最短距离即为所求。‌

# `第十九题 数字和为sum的方法数`

\###大概题意 给你一个容量为sum的背包，问你有多少种方法可以装满。

### 做法

动态规划之背包问题，f(i,j)定义为用前i个数字可以凑到j的方案数。如果我们不选第i个数 f[i][j]是不是就等于f[i-1][j],如果我们选择第i个数，f[i][j]就等于f[i-1][j-a[i]]，所以有了状态转移方程，f[i][j]=f[i-1][j]+f[i-1][j-a[i]]。我们发现二维f(i,j)有些冗余，由于递推关系，f(i,j)只与i−1*i*−1有关,所以我们可以优化掉第一维，f(j),代表装满j重量的方案数为f(j),初始化，f[0]=1,装满0重量的背包有一种方案，时间复杂度为O（N∗M）*N*∗*M*）。

```

# `第二十题 数论只会gcd`

### 大概题意

给定俩个数a,b，满足gcd(a+k,b+k)不互质，求出最小的k，如果找不到则输出-1

### 做法

当 b−a=1*b*−*a*=1 时 gcd(a+k,b+k)永远互质，这个时候输出-1。 gcd(a+k,b+k)*g**c**d*(*a*+*k*,*b*+*k*)<=><=>gcd(b−a,a+k)*g**c**d*(*b*−*a*,*a*+*k*)，设b−a*b*−*a*的因子为x，则最小的k则为x−(a*x*−(*a*%x)*x*)，可以直接枚举因数x，但是因数太多了，于是我们可以枚举质因数，可以用线性筛或者欧拉筛来筛质数。时间复杂度O(N∗log(N))*N*∗*l**o**g*(*N*))

```
#include<bits/stdc++.h>
using namespace std;
typedef long long ll;
mt19937 rnd(time(0));
#define int long long
typedef tuple<int,int,int> tp;
#define x first
#define y second
typedef pair<int,int> pii;
typedef pair<double,double> pdd;
constexpr int N=1000010,mod=1e9+7,inf=1e18;
constexpr double pi=3.1415926535897932384626;
const ll  P=rnd()%mod;
#define all(a) a.begin(),a.end()
#define get_count(x) __builtin_popcount(x)
int dx[]={0,1,0,-1,1,1,-1,-1,0};
int dy[]={1,0,-1,1,1,-1,-1,1,0};
int p[N],idx,st[N];
void get_p(int n){
    for(int i=2;i<=n;i++){
        if(!st[i]){
            p[idx++]=i;
            st[i]=i;
        }
        for(int j=0;p[j]*i<=n;j++){
            st[p[j]*i]=p[j];
            if(i%p[j]==0) break;
        }
    }
}
signed main(){
    cin.tie(nullptr)->sync_with_stdio(false);
    int __=1;
    cin >> __;
    get_p(N-1);
    while(__--){
        int a,b;
        cin >> a >> b;
        if(__gcd(a,b)!=1) cout << "0\n";
        else{
            int x=b-a,ans=inf;
            while(x!=1){
                int xx=st[x];
                ans=min(ans,xx-a%xx);
                x/=xx;
            }
            cout << (ans==inf?-1:ans) << '\n';
        }
    }
    system("color 04");
    return 0;
}
